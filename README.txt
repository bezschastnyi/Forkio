В проекте используется менеджер задач Gulp, который собирает, компилирует и минифицирует файлы в production версии. Файлы стилей написаны с помощью предпроцессора Sass, 
проект разбит на отдельные секции, которые были вставлены в основной файл html, стили также разбиты на отдельные файлы scss, которые собираются в основной. В проекте 
используюется адаптивная верстка. 
Проект выполнили:
 - Кобылянский Виталий:
Свертал блок Revolutionary Editor, секцию Here is what you get, секцию Fork Subscription Pricing
- Безсчасный Николай: 
Свертал шапку сайта с верхним меню. (включая выпадающее меню), свертал секцию People Are Talking About Fork
